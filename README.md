# 项目简介

## 核心功能
- 基于YOLOv8的叉车检测系统，支持检测'forklift'和'person'两类目标
- 提供900张标注数据集
- 支持图片、视频及摄像头三种输入方式进行检测
- 界面实时显示目标位置、目标总数、置信度等信息
- 支持批量检测并查看所有结果
- 支持检测结果保存
- 支持自定义置信度和IOU阈值
- 可选择显示标签和原图
- 提供多种检测模型选择

## 适用场景
- 工业场景中的叉车安全监控
- 仓储物流区域的人员与设备管理
- 需要实时检测叉车和人员存在的环境
- 批量处理图片或视频中的目标识别任务

## 技术特点
- 使用Pycharm+Anaconda或VSCode+Anaconda开发环境
- 依赖python3.9、opencv-python、PyQt5、ultralytics、torch1.9等技术栈
- 包含完整程序文件、UI界面源文件、数据集图片和运行教程
- 预训练模型可直接使用，环境配置成功后即可运行
